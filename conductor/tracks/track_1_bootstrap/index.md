# Track 1: Bootstrap

## Overview
This track covers the core initialization of the Synapse project, setting up the compilation pipelines across Web, Mac, Windows, and Linux, bootstrapping the `flutter_rust_bridge` v2 and Bevy dependencies, integrating Riverpod State Management with annotations, and building a responsive, highly visual 3D Editor Dashboard.
